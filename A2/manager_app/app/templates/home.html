Welcome! Manager

This ​app​ is developed for the ​manager to monitor and control the worker pool. To use, you can directly click on their function name in their descriptions or you can click them in the navigation bar.


You can press Worker List to enter the "Worker List" page. In this page, you can create or destroy instances. You can also monitor some detailed information of instances such as CPU utilization rate and HTTP request rate.

You can press Auto Scale to enter auto scale page. This page will first display the current autoscaling policy. Then, users can tuning their desired policy by adjusting the value of CPU grow threshold, CPU shrink threshold, grow ratio and shrink ratio. Then just click “update” button to actually apply the policy.

Or you can press ELB DNS to return to user app directly.

You can also press Delete to delete the application data stored on the database. as well as all images stored on S3.

You can press Stopping to terminates all the workers and then stops the manager itself. {% endblock %}
